# Passive vs. Active Mutual Fund Investment

**Caden Eskelsen | May 2025**

---

## Overview

This repository contains a research paper analyzing whether **actively managed mutual funds outperform passive investment strategies** after accounting for fees and risk. Using empirical methods and financial theory, the paper evaluates if active management delivers meaningful excess returns.

---

## Methodology

* **Model:** Single Index Model and CAPM
* **Data:** 50 mutual funds, 50 months of monthly returns
* **Benchmark:** Market index (e.g., S&P 500)
* **Tests:** White (heteroskedasticity), Durbin-Watson (autocorrelation), RESET (specification)

---

## Key Findings

* No statistically significant **alpha** for most funds
* Returns are **similar to the market**, not superior
* Some funds exhibit **lower volatility**, but not higher returns
* **Fees reduce net performance**, limiting benefits of active management

---

## Conclusion

Passive investing provides comparable returns **without the added costs**, making it a more effective long-term strategy for most investors.
